# Software-Requirements-Specification-Generation-from-User-Requirements-using-NLP
This project focuses on transforming unstructured user requirements into Structured Natural Language (SNL) using Natural Language Processing (NLP) techniques. It aims to reduce ambiguity, inconsistency, and incompleteness in software requirement gathering, thereby saving time, effort, and cost during the software development lifecycle.
#Key Features
Converts raw user requirements into structured specifications
Identifies condition statements (If, When, While, Where)
Detects system actions and actor-initiated actions
Handles other actions using generic templates for complete coverage
Evaluates performance against existing state-of-the-art NLP approaches
Technical Details
Desktop application (Python,NLP)
Uses multiple NLP techniques to process and structure requirements
Focuses on improving the software requirement analysis phase
